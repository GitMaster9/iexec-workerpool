1. pokrenuti iexec CLI (wallet)
iexec init --skip-wallet

2. provjeri chain.json (bellecour mora biti pod default)
{
  "default": "bellecour",
  "chains": {
    "mainnet": {},
    "bellecour": {}
  }
}

3. kreiraj wallet (kreiraj password, npr: password123core)
iexec wallet create --keystoredir ./

4. preimenuj kreirani wallet (npr. worker_wallet.json)

5. kreiraj Docker network (ako worker mora biti na istoj mre≈æi kao scheduler core)
docker network create iexec_network

6. pokreni Docker kontejnere iz worker foldera
cd worker
docker compose up -d
